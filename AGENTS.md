# MaiZang Engine Agent Contract

These instructions apply to the entire repository.

## Required Context

Before changing code, tools, scenes, tests, or workflow:

1. Read `.codex/skills/maizang-engine-godot/SKILL.md`.
2. Read `wiki/Current-State.md`.
3. Read the relevant architecture or pipeline Wiki page.
4. Inspect the current Git status and preserve unrelated user changes.

## Change Protocol

Every functional change must include all of the following in the same commit:

1. The implementation and focused tests.
2. A semantic update to the relevant page under `wiki/`.
3. An entry in `wiki/Change-Log.md`.
4. A refreshed `wiki/Current-State.md` and Skill project state, generated by:

   ```powershell
   powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\tools\update_project_memory.ps1 -Stage
   ```

5. Verification appropriate to the blast radius.

The versioned pre-commit hook rejects functional commits that do not update
`wiki/Change-Log.md`. Configure it with `tools/setup_repository.ps1`.

## Asset Boundary

Do not commit or push ROMs, DSPRE exports, Pokemon models, textures, maps,
audio, text archives, or generated derivatives. The following paths must stay
ignored:

- `generated/`
- `.work/`
- `new-game-project/assets/platinum/`
- `new-game-project/captures/`

Keep scripts and manifests schemas reproducible without embedding proprietary
content. Validate ignored files with `git status --ignored` when changing the
asset workflow.

## Runtime Invariants

- Godot target: 4.7 stable, compatibility renderer.
- Logical viewport and window: `256 x 192`, fixed 4:3 NDS screen.
- Physics cadence: fixed `60 Hz`.
- Player movement is cardinal-only on a 1-world-unit grid (16 source pixels).
  Grid centers are at half-integer X/Z coordinates. Platinum's 30 Hz walk/run
  actions take 8/4 source frames, mapped to 16/8 Godot physics ticks per cell.
  A stationary direction change takes 6 Godot ticks.
- The camera uses yaw `0`, downward pitch `50`, and wheel step `5`. It starts
  orthographic at size `11.24` and distance `16`; `F1` toggles the distance-8,
  FOV-75 perspective debug view.
- Matrix coordinate unit: 32 world units per map block.
- Matrix altitude unit: 0.5 world units.
- Imported model scale: `1 / 16`.
- Streamed assets must load asynchronously without sub-threaded import.
- Runtime code must not mutate shared `ArrayMesh` materials.
- Distant chunks and their strong `PackedScene` references must be released.

## Git Workflow

Use one focused commit for each completed change. Prefer:

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\tools\commit_project_change.ps1 `
  -Message "Short imperative commit subject" `
  -Summary "What changed and why"
```

Do not rewrite published history unless the user explicitly requests it.
